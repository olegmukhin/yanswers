Чего вы ожидаете от участия в Школе?

Когда мне было 12 лет, я создал мою первую страницу. Все началось с простых страниц, потом с появлением Flash я начал экспериментировать с созданием игр. В будущем мое желание к получению новых знаний в сфере интернета постаянно возрастало. Я самостоятельно изучал новые языки программирования, в результате чего приходилось искать огромное количество информации о самых эффективных методах выполнения задач в интернете. Из-за этого, у меня есть много знаний, но а также имеются некоторые пробелы. После университета я понял, что меня привлекают именно способы и технологии, которые мы используем, чтобы работать с компьютерами, и поэтому я хочу научиться всем тонкостям создания интерфайсов у профессионалов и лидеров в этой индустрии, чтобы по настоящему начать мою профессиональную карьеру в этой сфере.

Опыт разработки

Расскажите нам о вашем опыте разработки. Ничего не упускайте: это может быть как серьезный интерфейс, так и просто домашняя страничка.

Сложно перечислить все страницы и сайты над которыми я работал. В раннем возрасте я вместе с моим школьным другом создали сайт, для которого мы разрабатывали Flash-игры. С начала игры были очень простые, например крестики и нолики, но со временем появились более технологически-интенсивные игры (в основном с гоночными машинами), которые использовали сложные алгоритмы для физики.

В следующей фазе моей жизни (в старших классах) я создавал сайты для друзей и компаний, иногда бесплатно, а иногда за деньги. Это было более 4-x лет назад, так что большинство сайтов уже не функциональны, но осталось всего два, которые можно показать. Один из них это сайт-тренер для Британской Ассоциации по Дзюдо (использует LAMP и достаточно плохой дизайн, но весь код я написал с нуля), а другой это простой сайт с элементом E-Commerce, который я создал для маленького британского производителя часов (опять LAMP, с подключением к Flickr для фото галереи и опять все кроме интеграции Paypal и Google Checkout с нуля).

В университете мне пришлось создавать много страниц, так как это было частью моего курса, но могу отметить всего один проект. Один из самых больших проектов, который был задан моей команде в университете. Это система по созданию расписания для университета. Это система должна была получать информацию, как желаемое время проведения лекции, нужный размер зала, необходимое оборудование (проектор и т.д.) и расположение в университете. А в итоге предлагать возможные варианты, которые не забронированы на данное время, и если надо находить другие варианты по расстоянию от желаемого местоположения и всех вышеперечисленных факторов. Этот проект был выполнен используя JS / LAMP, и потом в ASP.NET (как требовала спецификация задания).

Во время работы в IBM, я предложил создать CMS систему для поддержки большого внутреннего сайта, который мой департамент обновлял вручную (сайт в конце концов попадал в IBM PartnerWorld). Моему менеджеру понравилась эта идея, и в течение 12 месяцев я создал новую систему используя jQuery и JSON для передачи данных между базой данных и браузером пользователя, а также уже очень знакомую мне конфигурацию LAMP. Я использовал слегка другой подход и пытался приддерживаться мышлению WYSIWYG. Идея была создать страницу, где можно кликнуть на любой кусок текста для редактирования (то есть текст приврашаеться в редактируемое поле без визуальных изминений для пользователя кроме появления курсора), и где практически все элементы можно переместить используя методику drag and drop. В течение года я закончил весь функционал, который был оговорён, и передал проект на поддержку другим сотрудникам компании перед моим уходом.

За последний год я экспериментировал с новыми для меня технологиями. Я приложил руку к созданию несколько презентаций на базе Ventuz. А также я разработал регистрационную систему оптимизированную под мобильные устройства на стенде «Сколково» на «Роснанотехфоруме». Сейчас я нахожусь, в процессе создания приложения на платформе iOS для британского журнала. Цель приложения - это дать возможность пользователям рассматривать и листать изображения в более удобном и высококачественном формате, которые журнал публикует в своем блоге.

/**
 * Создает экземпляр Машины
 * @this {Car}
 * @param {string} manufacturer Производитель
 * @param {string} model Модель
 * @param {number} year Год производство
 */
function Car(manufacturer, model, year) {
    this.manufacturer = manufacturer;
    this.model = model;
    this.year = typeof year !== 'undefined' ? year : new Date().getFullYear();
}

Car.prototype.toString = Car.prototype.getInfo = function() {
    return this.manufacturer + ' ' + this.model + ' ' + this.year;
};
Car.prototype.getDetailedInfo = function() {
    return 'Производитель: ' + this.manufacturer + '. Модель: ' + this.model + '. Год: ' + this.year;
};

var bmw = new Car("BMW", "X5", 2010),
    audi = new Car("Audi", "Q5", 2012),
    toyota = new Car("Toyota", "Camry");

/**
 * Создает экземпляр Автосалона
 * @this {CarDealer}
 * @param {string} name Название автосалона
 */
function CarDealer(name) {
    this.name = name;
    this.cars = [];
}

var yandex = new CarDealer('Яндекс.Авто');

CarDealer.prototype.add = function() {
    for (var i = 0; i < arguments.length; i++) {
        this.cars.push(arguments[i]);
    }
    return this;
};

yandex
    .add(toyota)
    .add(bmw, audi);

/**
 * Установить цену на машину
 * @param {string} car идентификатор машины
 * @param {string} price стоимость
 */
// идентификатор машины составляется следующим образом "производитель модель год"
// стоимость машины может быть задана в двух валютах: йена и евро.
CarDealer.prototype.setPrice = function(car, price) {
    for(var i = 0; i < this.cars.length; i++) {
        if(this.cars[i].getInfo() == car) {
            this.cars[i].price = price;
        }
    }
    return this;
};

yandex
    .setPrice('BMW X5 2010', '€2000')
    .setPrice('Audi Q5 2012', '€3000')
    .setPrice('Toyota Camry 2012', '¥3000');

    var results = [];

CarDealer.prototype.listInRubles = function() {
    var all = [];
    for(var i = 0; i < this.cars.length; i++) {
        var info = this.cars[i].getInfo();
        if(this.cars[i].price !== 'undefined') {
            var price = this.cars[i].price;
            if(price.charAt(0) == '€') {
                var rubles = price.substr(1) * 39.7527;
            } else {
                var rubles = price.substr(1) * 40.3451;
            }
            info += ' ' + rubles.toFixed(2) + ' руб.';
        }
        all.push(info);
    }
    return all.join(', ');
};

#!/usr/bin/env perl

use strict;
use Getopt::Long;
use Pod::Usage;

my $help;
my $message = '';
my $v;

@ARGV and GetOptions(
    "h" => \$help,
    "m:s" => \$message,
    "v" => \$v,
) or pod2usage(1);
pod2usage(-verbose => 2, -exitval => 2) if $help;

my $count = 0;

foreach my $a (@ARGV) {
    if($v) {
        print($a . "\n");
    }
    $count++;
}

if($message ne "") {
    print($message . "\n");
}

print($count . "\n");

__END__

=head1 SYNOPSIS

#!/usr/bin/env python

import argparse

parser = argparse.ArgumentParser(description='Print the number of arguments.')
parser.add_argument('arguments', metavar='ARG', type=str, nargs='*', help='some arguments')
parser.add_argument('-m', dest='message', default='', help='custom message')
parser.add_argument('-v', action='store_true', help='verbose mode')

args = parser.parse_args()
count = 0

for a in args.arguments:
    if args.v:
        print a
    count += 1

if args.message != '':
    print(args.message)

print(count)
